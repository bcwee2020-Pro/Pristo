# Luxury IdeaGen AI

Premium UI version of the Content Idea Generator micro-SaaS.

## Setup
1. npm install
2. Add OPENAI_API_KEY in environment (.env.local or Vercel)
3. npm run dev

## Notes
- Uses Next.js app router (app/).
- Tailwind CSS is configured (devDependencies included). Run `npx tailwindcss init -p` if needed.
